# Baidu-Wenku-Parse
一个简单的baidu wenku解析，可讲链接中的文档以txt的格式保存至本地（目前仅支持doc格式的文档解析）

## 如何使用
1. 安装 *requirements.txt*
```
pip install -r requirements.txt
```
2. 运行 *docParser.py*
```
python3 docParser.py 
```
3. 输入url
4. 解析的文件会以txt的格式将会生成于 *../data* 中
